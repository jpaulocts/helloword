# Hello World with Next.js - Beginning My Full-Stack Journey

## Overview

As an aspiring full-stack software engineer, I believe in mastering modern frameworks and tools that shape the industry. This "Hello World" project built with Next.js represents my commitment to learning cutting-edge web development technologies. Throughout this course, I aim to deepen my understanding of React frameworks, server-side rendering, and the complete development workflow from local setup to production deployment.

This software is a modern web application built with Next.js 15, featuring both Server and Client Components. Beyond the traditional "Hello World" message, it demonstrates:
- **Server-side rendering** with Next.js App Router
- **Client-side interactivity** using React hooks
- **Responsive design** with Tailwind CSS
- **TypeScript** for type safety
- **Modern UI patterns** with gradient backgrounds and interactive elements

My purpose in creating this software is to establish a solid foundation with Next.js before tackling more complex modules. By setting up a complete development environment with TypeScript, ESLint, and Tailwind CSS, I'm preparing myself for professional-grade application development.


## Development Environment

### Tools Used
- **IDE**: Visual Studio Code with extensions for TypeScript, Tailwind CSS, and ESLint [citation:1]
- **Version Control**: Git with GitHub for repository management
- **Package Manager**: npm (comes with Node.js)
- **Browser**: Chrome DevTools for debugging and testing

### Technology Stack
- **Framework**: Next.js 15 with App Router [citation:2]
- **Language**: TypeScript for enhanced type safety and developer experience
- **Styling**: Tailwind CSS for utility-first styling
- **Linting**: ESLint with Next.js core web vitals configuration
- **Runtime**: Node.js 18+ (required for Next.js development)

### Why This Stack?
- **Next.js** provides both server-side rendering and static site generation, offering flexibility for different project needs
- **TypeScript** catches errors early and provides excellent IDE support
- **Tailwind CSS** enables rapid UI development without context-switching
- **App Router** (new in Next.js 13+) offers intuitive file-based routing and React Server Components [citation:2]
